# Personal Website

My personal developer website to showcase skills, projects, and experience. Built with Python (Flask), TypeScript, and Terraform for cloud deployment.

Accessible at [josephcicalese.com](https://josephcicalese.com)

## Features

- Flask backend with modular app structure
- TypeScript/JavaScript frontend assets
- Project and course showcase
- User authentication
- Responsive design
- Infrastructure as Code with Terraform (GCP)
- Nginx reverse proxy and SSL via Cloudflare

## Tech Stack

- Python 3
- Flask
- TypeScript & JavaScript
- HTML/CSS
- Terraform (Google Cloud Platform)
- Nginx
- Cloudflare

## Directory Structure

- `app/` - Flask application modules, assets, and templates
- `ts-src/` - TypeScript source files
- `terraform/` - Infrastructure configuration
- `requirements.txt` - Python dependencies
- `package.json` - Node dependencies
